축 3
핵심 모트 (Moat)

Strategic Role
교사에게 필요한 것을 제공하여,
교사가새로운 가치를 창출할 수 있게
네비게이터(Navigator) 역할

01
데이터 기반 시각화 대시보드

02
AI 상담 코칭 에이전트

→ 교사의 상담 역량 증폭
03
SEL 개인 맞춤형 자료 추천
사회정서교육(SEL) 자료를 학생별 검사 결과에 기반해 자동 추천. 상담과 교육의 연계 실현
→ 교사-학생 상호작용 유도

🔄 데이터 루프 기여: 학습 이력 축적
주요 기능
01
3분 실시간 라이브 게임
수업 중 즉시 활용 가능한 라이브 퀴즈 게임. 교사가 빠르게 학생의 이해도를 파악하는 수업 도구
→ 수업 활성화 + 즉각 피드백
02
개인 맞춤형 복습 게임
학습 이력 데이터를 기반으로 개별 학생의 취약점에 맞는 문항 자동 추천. AI 기반 적응형 학습
→ 교사의 수업안 설계 부담 해소
03
교사 학습지도 개입 데이터
게임 참여 데이터를 집계하여 교사에게 "지금 개입이 필요한 학생"을 알림. 교사 역할을 강화
→ 학생 참여가 교사 인사이트로 전환

01

학생별 종합 데이터를 AI가 분석하여 교사에게 맞춤 인사이트 제공. 상담·수업 개선에 활용
→ 교사의 의사결정 지원
03
교사 주도 서비스 구성
교사가 필요한 서비스를 직접 조합·활용하여 수업을 설계. 플랫폼 내 활동을 자연스럽게 유도
→ 교사 중심 생태계 형성
